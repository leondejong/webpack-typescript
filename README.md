# Webpack TypeScript Sass Starter Kit

Install: `yarn`  
Start: `yarn start`  

Visit: `http://localhost:8080/`  

Build Dev: `yarn dev`  
Build Prod: `yarn prod`  

Lint Code: `yarn lint-code`  
Lint Style: `yarn lint-style`  

Test: `yarn test`  

## Recommended Editor Extensions

TypeScript  
Sass  
Prettier  
ESLint  
Stylelint
